# GoFare (Public Shell Site)

This repository hosts the public informational shell for **GoFare** (GitHub Pages).

GoFare is a flight metasearch experience:
- Compare travel options
- Continue to airlines/partners to complete booking
- GoFare does not issue tickets or collect traveler payments on this site
